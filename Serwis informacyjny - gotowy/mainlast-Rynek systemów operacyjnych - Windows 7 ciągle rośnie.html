    • Rynek systemów operacyjnych - Windows 7 ciągle rośnie

    Autor: Kamil Zając | 09-04-2015

    Każdy producent oprogramowania liczy na to, że klienci czym prędzej przesiądą się na najnowszą wersję danej aplikacji lub systemu operacyjnego. Czasami użytkownicy okazują się jednak bardzo oporni we wprowadzaniu zmian w swoich komputerach i korzystają ze starszych produktów przez długie lata. Microsoft od zawsze boryka się z problemem dominacji wcześniejszych systemów Windows, co jest widoczne szczególnie po zakończeniu wsparcia dla wersji XP wydanej w 2001 roku - teoretycznie, udział tego systemu powinien bardzo szybko spadać w relatywnie krótkim okresie czasu, jednak rzeczywistość jest zupełnie inna.

    Co jakiś czas do sieci trafiają statystyki rynku systemów operacyjnych przedstawiające procentowy udział wielu wersji. Naprawdę, nie trzeba być ekspertem, aby przewidzieć zwycięzce kolejnego rankingu. W marcu 2015 roku sytuacja przedstawiała się następująco:

    1. Windows 7 - 58,04%

    2. Windows XP - 16,94%

    3. Windows 8.1 - 10,55%

    4. OS X 10.10 Yosemite - 3,96%

    5. Windows 8 - 3,52%

    6. Linux - 1,5%

    Na samym szczycie nie mogło zabraknąć siódemki, której udziały w marcu wzrosły z 55,99% do 58,04%. Na drugiej pozycji dochodzi oczywiście do ciągłych strat - popularność Windows XP spadła z 19,15% do 16,94%. Najnowszy dostępny system operacyjny od Microsoftu (Windows 8.1) rośnie bardzo powoli, gdyż udziały z ubiegłego miesiąca na poziomie 10,49% wzrosły zaledwie do 10,55%. Kolejne pozycje zajmuje OS X 10.10 Yosemite (3,96%), Windows 8 (3,52%) oraz Linux (1,5%). Jeśli chodzi o testowy Windows 10 Technical Preview, to obecnie korzysta z niego około 0,09% użytkowników.

    Windows XP na razie nie oddaje swojej pozycji najnowszej wersji 8.1.
